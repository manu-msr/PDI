¿Qué es un procesador de Imágenes?

Es un programa que aplica un conjunto de técnicas a imágenes ditiales con el objetivo de mejorar la calidad o falicitar la búsqueda de información.

¿Cómo se aplica un filtro?

Una imagen está formada por pixeles y cada pixel, en esencia está formado por tres componentes de color: Una roja, una verde y una azul (R,G,B pos sus iniciales en inglés). Al combinar estas componentes podemos obtener prácticamente cualquier color. Al aplicar un filtro de modifican estas componenten con el fin de mejorar la calidad de la imagen o facilitar la búsqueda de información.

¿Qué filtros tiene implementados este programa?

Este programa incluye filtros sencillos para modificar el color, filtros de escala de grises, para girar imágenes, aplicar reducir, crear favicones, fotos recursivas, foto mosaicos, óleo y demás filtros artísticos.
